# نظام التعرف على بصمة الوجه

تطبيق Android متكامل للتعرف على الوجه مع دعم كامل للغة العربية.

## ✨ المميزات
- التعرف على الوجه في الوقت الفعلي
- تسجيل وجوه جديدة بزوايا متعددة
- دعم استيراد الصور من المعرض
- واجهة عربية كاملة
- إعدادات مرنة (عتبة التعرف، تحسين الصور، الأصوات)
- تسجيل الحضور والانصراف
- قاعدة بيانات SQLite محلية

## 📋 المتطلبات
- Android 5.0 (API 21) أو أحدث
- كاميرا خلفية أو أمامية
- مساحة تخزين كافية للصور والبيانات (100MB على الأقل)

## 📱 التثبيت
1. تحميل ملف APK من صفحة Releases
2. تمكين التثبيت من مصادر غير معروفة
3. تثبيت التطبيق

## 🎯 الاستخدام
1. افتح التطبيق
2. وجه الكاميرا نحو الوجه
3. سيتم التعرف تلقائياً على المسجلين
4. للتسجيل الجديد، اضغط على "تسجيل جديد"
5. التقط 5 صور من زوايا مختلفة
6. أدخل البيانات الشخصية

## 📁 الملفات المطلوبة على الهاتف
