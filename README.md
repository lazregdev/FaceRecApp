# نظام التعرف على بصمة الوجه

تطبيق Android للتعرف على الوجه مع دعم كامل للغة العربية.

## المميزات
- التعرف على الوجه في الوقت الفعلي
- تسجيل وجوه جديدة بزوايا متعددة
- دعم استيراد الصور من المعرض
- واجهة عربية كاملة

## المتطلبات
- Android 5.0 (API 21) أو أحدث
- كاميرا خلفية أو أمامية

## التثبيت
1. تحميل ملف APK من إصدارات GitHub
2. تمكين التثبيت من مصادر غير معروفة
3. تثبيت التطبيق

## المطور
لزرق عبدالقادر - الأغواط، الجزائر
